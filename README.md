# FlashDeck

The goal of this project was to create a flash card site for my personal use.

## Technologies

- ReactJS
- Vite
- Vercel
- [PandaCSS](https://panda-css.com/)
- [Shadow Panda](https://shadow-panda.dev/)

## Features

- Create flash cards
- Save flashCards on server
- Study flashCards
- Share FlashDecks
